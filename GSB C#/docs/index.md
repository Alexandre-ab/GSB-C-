# GSB WinForms — Documentation

Bienvenue dans la documentation du projet GSB. Cette documentation est générée avec DocFX pour offrir une navigation similaire à l’exemple de gsbMonolith DAO (voir <https://ryuksylux.github.io/gsbMonolith/api/gsbMonolith.DAO.html>). Vous trouverez :

- Une présentation fonctionnelle et technique dans la section « Présentation ».
- Une référence API générée automatiquement à partir du code C# (DAO, modèles, formulaires et utilitaires).
- Des instructions pour générer et publier le site statique.

Utilisez le menu de gauche pour naviguer entre les pages et la recherche intégrée pour retrouver rapidement une classe ou une méthode.
